Tópico 2: Working with java data types

- A classe Boolean possui dois construtores:
  Boolean(String arg) ou Boolean(true/false)
  
- Na declaração primitivos, não pode haver 0. Exemplo: int num = 09 //inválido

- Na declaração de primitivos, podem ser colocado o caracter _ entre os dígitos / Exemplo: int num 123_456; //válido

- Nunca podemos chamar um método que recebe array como se ele fosse varargs

- StringBuilder x StringBuffer / Builder é synchronized e é mais rápido também

- Classes que não podem ser extendidas:
    - Classes de tipo primitivo
    - Classe System
    
- byte: de -128 127

- o char é somente positivo

- byte -> short -> int -> long

- float -> double
